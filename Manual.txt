Каждый игровой объект имеет свой Target. Target строится следующим образом:
Если объект является Board (игровое поле) его таргет Board.
Если объект является Tile (игровое поле) его таргет Board.Tile.X. Где X - номер Tile.
Если объект является Space (игровое поле) его таргет Board.Tile.X.Space. Где X - номер Tile, Space которого мы желаем рассмотреть.
Если объект является Planet (игровое поле) его таргет Board.Tile.X.Planet.Y. Где X - номер Tile, Space которого мы желаем рассмотреть, а Y - номер Planet тайла X.
Если объект является Player (игровое поле) его таргет Player.X. Где X - номер Player.
Если объект является Army (игровое поле) его таргет Player.X.Army. Где X - номер Player.
Если объект является Ship (игровое поле) его таргет Player.X.Army.Ship.Y. Где X - номер Player. Где Y - номер Ship.
Если объект является Infantry (игровое поле) его таргет Player.X. Infantry.Ship.Y. Где X - номер Player. Где Y - номер Infantry.

На запрос  “Input playerX name”, где X - число, номер игрока введите слово без пробелов - игровой ник игрока X.
На запрос “Input race name” введите расу игрока, имя которого было введено перед этим.
На запрос “Input Igor's strategy command name” введите “pick X”, где X номер стратегии, которую вы хотите выбрать.
На запрос “Input Igor's action command name” вызовите одну из следующих команд:
add-unit - команда создающая корабль. При ее вызове появится запрос “Input unit name”. Введите тип юнита, который вы хотите создать (типы юнитов: …). Далее появится вопрос “Input tile object target”. На него надо ввести Target объекта в котором вы хотите разместить ваш юнит (объeкт должен являться Tile object, т. е. либо Space, либо Planet).
move - команда передвигающая корабли. Появится запрос “Input active tile object target”. На него введите таргет Tile object, в который вы хотите передвинуть юниты, удалив из начала Board.

Во время любого запроса команды можно вызвать команду game, которая переведет Вас в режим глобальных комнад. Бывает 2 глобальные команды:
1) view - команда выводящая объекты. Далее необходимо ввести Target объекта.
2) exit - команда, завершающая игру.
